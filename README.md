## Development

1. Clonar proyecto

2. Copiar `.env.template` y renombar a `.env` y agregar las variables de entorno

3. Ejecutar

```
npm install
```

4. levantar base de datos

```
docker-compose up -d
```

5. Levantar el backend de Nest

```
npm run start:dev
```

6. Para ver los end points

```
localhost:3000/api
```
